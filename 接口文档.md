# 登录模块

## POST 登录

POST /login

> Body 请求参数

```json
"{account: \"yyy\", password: \"yyy\"\r\n}"
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 否 |none|
|» account|body|string| 是 |none|
|» password|body|string| 是 |none|

> 返回示例

> 成功

```json
{
  "code": 0,
  "message": "成功",
  "data": {
    "Oauth-Token": "minim deserunt officia"
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|成功|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none|响应码|0-成功，1-失败|
|» message|string|true|none|返回信息|成功|
|» data|object|true|none|返回数据|数据中包含token|
|»» Oauth-Token|string|true|none||token|
